# Khaled Ben Yahya — Portfolio

Personal portfolio site built with React + Vite.
Showcases selected projects, technical writing, and contact info.

🔗 **Live:** https://khaledbenyahya.com/

## Tech Stack

- React 19
- Vite
- Vanilla CSS3 (custom design system, no component library)

## Getting Started

```bash
npm install
npm run dev
# Open http://localhost:5173
```
